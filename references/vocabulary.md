# Vocabulary reference

Words and phrases that spiked in published text after LLMs arrived. Kobak et al. (arXiv:2406.07016) measured the excess across 15M+ PubMed abstracts — words like "delves", "showcasing", and "underscores" appeared many times more often in 2024 than their 2010–2022 trend predicted. Follow-up work (arXiv:2502.09606) shows the fingerprint shifts between model generations, so treat this list as a living document.

Rules of use:

- Replace, don't thesaurus. The fix for "leverage" is "use", not "utilize harness capitalize on".
- If the user's own text or the domain genuinely uses a word ("robust statistics", "attack surface landscape"), keep it. Accuracy beats avoidance.
- A single hit is fine. Three in a paragraph is a signature.

## Verbs

| Banned | Use instead |
|---|---|
| delve into | look at, examine, dig into |
| leverage | use |
| utilize | use |
| harness | use |
| foster | build, encourage |
| showcase | show |
| underscore | show, stress (or state the fact plainly) |
| highlight (reflexive) | say it directly |
| elevate | improve, raise |
| embark on | start, begin |
| navigate (metaphorical) | handle, deal with, work through |
| unlock | make possible, enable (or name the concrete gain) |
| empower | let, enable, help |
| streamline | simplify, speed up |
| bolster | support, strengthen |
| facilitate | help, ease |
| garner | get, earn, win |
| revolutionize | change (state the actual change) |
| boast(s) | have, has |
| encompass | include, cover |
| align (vague) | agree, match, fit |

## Stock phrases

Cut these entirely; they carry no information.

- in today's fast-paced world / digital age / ever-changing landscape
- at the end of the day
- when it comes to
- deep dive / dive deep into
- take it to the next level
- in the realm of
- it's worth noting that / it's important to note that
- needless to say
- a wide range of / a diverse array of
- best practices (unless naming specific ones)
- moving forward (as filler)
- last but not least
- first and foremost
- in order to → to
- due to the fact that → because
- the process of X-ing → X-ing

## Puffery

Wikipedia's AI-cleanup project flags these as promotional tone regardless of who wrote them. Describe what the thing does, not how impressive it is.

- renowned, esteemed, acclaimed, celebrated
- rich cultural heritage, rich history
- nestled (in the heart of)
- stands as / serves as a beacon of
- world-class, premier, leading (unverifiable)
- award-winning (unless naming the award)

## Connectives to ration

Not banned, but AI leans on them far more than humans do. More than one or two per page reads as generated.

- Additionally / Furthermore / Moreover → often just start the next sentence
- However (sentence-initial, repeatedly) → vary or drop
- Overall / Ultimately / In essence → usually precedes an echo; cut the sentence
- Notably / Interestingly / Importantly → delete; the content shows it or it doesn't
